CSS PROPERTIES WITH DETAILED EXPLANATIONS
==========================================

1. color:
   - Description: Sets the text color.
   - Example Values: red, blue, #000000, rgb(255, 0, 0)
   - Explanation: Determines the color of the text content in an element.

2. background-color:
   - Description: Sets the background color of an element.
   - Example Values: yellow, #ffcc00, rgba(0, 255, 0, 0.5)

   - Description: Specifies the size of the font.
   - Example Values: 16px, 1em, large, small
   - Explanation: It controls the size of the text, making it larger or smaller.

4. font-family:
   - Description: Defines the typeface (font style) for the text.
   - Example Values: Arial, Verdana, "Times New Roman", sans-serif
   - Explanation: Helps to improve readability and style of text.

5. text-align:
   - Description: Aligns text within an element.
   - Example Values: left, right, center, justify
   - Explanation: Controls how the inline content (like text) is aligned.

6. margin:
   - Description: Sets the space outside the border of an element.
   - Example Values: 10px, 1em, auto
   - Explanation: It creates space between elements (outer spacing).

7. padding:
   - Description: Sets the space between the content and the border of an element.
   - Example Values: 10px, 5%, 1em
   - Explanation: It creates inner spacing inside the element.

8. border:
   - Description: Sets the border around an element.
   - Example Values: 1px solid black, 2px dashed red
   - Explanation: Defines the width, style, and color of the border.

9. width:
   - Description: Sets the width of an element.
   - Example Values: 100px, 50%, auto
   - Explanation: Determines how wide the element will be displayed.

10. height:
    - Description: Sets the height of an element.
    - Example Values: 200px, 80%, auto
    - Explanation: Controls the vertical size of an element.

11. display:
    - Description: Specifies the display behavior of an element.
    - Example Values: block, inline, inline-block, none
    - Explanation: Determines how the element appears in the layout.

12. position:
    - Description: Sets how an element is positioned in the document.
    - Example Values: static, relative, absolute, fixed
    - Explanation: Controls the positioning method used to place an element.

13. float:
    - Description: Positions an element to the left or right.
    - Example Values: left, right, none
    - Explanation: Often used to wrap text around images or elements.

14. clear:
    - Description: Specifies what elements can float beside the cleared element.
    - Example Values: left, right, both, none
    - Explanation: Used to prevent elements from wrapping around floated elements.

15. overflow:
    - Description: Controls what happens to content that overflows the box.
    - Example Values: visible, hidden, scroll, auto
    - Explanation: Manages excess content inside elements.

16. visibility:
    - Description: Specifies whether an element is visible.
    - Example Values: visible, hidden
    - Explanation: Hides an element without removing it from the document layout.

17. opacity:
    - Description: Sets the transparency level of an element.
    - Example Values: 0 (transparent), 1 (opaque), 0.5
    - Explanation: Useful for fading effects or overlays.

18. z-index:
    - Description: Sets the stack order of an element.
    - Example Values: 1, 10, 100
    - Explanation: Determines which element appears in front when overlapping occurs.

19. line-height:
    - Description: Sets the height between lines of text.
    - Example Values: normal, 1.5, 20px
    - Explanation: Improves text readability by controlling spacing between lines.

20. text-decoration:
    - Description: Adds decoration to text.
    - Example Values: none, underline, overline, line-through
    - Explanation: Used for styling text with lines above, below, or through it.
    21 cursor:
    - Description: Specifies the type of cursor to be displayed when hovering over an element.
